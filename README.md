# 急診 CP-SAT 排班器

本機執行的急診醫師白／夜班排班工具。上傳既有 Excel 後，程式會先計算班數配額，再使用 CP-SAT 尋找符合硬性規則的最佳班表。

## 第一次使用

先確認已安裝 Python 3.10 以上，然後在命令提示字元執行：

```powershell
pip install streamlit ortools openpyxl python-docx
```

之後直接雙擊 `排班器.py`。瀏覽器會開啟本機排班頁面，CMD 視窗需保持開啟；關閉它會停止排班器。

## 使用流程

1. 選擇 `.xlsx` 預班／預假檔。
2. 確認班數分配預覽。
3. 按「產生 CP-SAT 班表」。
4. 在每日班表、醫師月曆、結果驗證之間切換。
5. 可按「換一個同品質最佳班表」尋找其他相同目標值的方案；箭頭可切回已暫存方案。
6. 下載 Excel 班表或 Word 值班表。

## Excel 必要工作表

- `表單回應`
- `急專醫師名單`
- `支援醫師名單`
- `特殊日期設定`

## 已實作的重要規則

- 每日白班、夜班各一位醫師。
- 預假不可排班；夜班後隔日不可白班。
- 急專醫師任一連續六日最多工作五日。
- 急專預班與外援指定班必排。
- 支援醫師總班數與假日配額精確符合設定。
- 急專假日班平均分配；支援醫師平日只從填寫的可上班時段選擇。
- 盡量避免 24 小時班，並偏好連上 3–4 天。

排班前可用「24 小時班容忍度」調整此偏好。這五個檔位使用不同的全月軟目標，因此會實際改變最佳解：`0` 強烈避免、`25` 約 1 個、`50` 約 2 個、`75` 約 4 個、`100` 約 6 個且主動偏好以 24 小時班換取較完整休息／長假。這是軟目標，硬性規則與既有指定班優先；預假連續 7 天以上的長假醫師，只放寬該醫師的 24 小時班限制。

## 結果閱讀

「醫師月曆」右側可點選醫師，該醫師所有班別會標亮。結果驗證頁會檢查目標班數、預假、夜接白、急專六日規則、指定班與支援配額。
